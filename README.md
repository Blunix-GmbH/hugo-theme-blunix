# Hugo Theme Blunix

A professional, clean Hugo theme designed for consulting and service-based businesses. Features multilingual support, responsive design, and modern best practices.

## Features

- Responsive, mobile-first design
- Multilingual support (i18n)
- SEO optimized
- Tailwind CSS for styling
- Custom blocks and shortcodes
- Clean, semantic markup

## Installation

### As a Git Submodule (Recommended)

```bash
git submodule add git@github.com:YOURUSER/hugo-theme-blunix.git themes/hugo-theme-blunix
```

### Manual Installation

Clone this repository into your Hugo site's `themes` directory:

```bash
cd your-hugo-site
git clone git@github.com:YOURUSER/hugo-theme-blunix.git themes/hugo-theme-blunix
```

## Configuration

Add the theme to your site's `config.toml` or `config/_default/config.toml`:

```toml
theme = "hugo-theme-blunix"
```

See the `exampleSite` directory for a complete configuration example.

## Development

The theme includes:
- `layouts/` - Template files
- `assets/` - CSS and image assets
- `static/` - Static files (fonts, JS, compiled CSS)
- `i18n/` - Translation files
- `archetypes/` - Content templates
